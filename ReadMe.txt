To setup your environment for the twitter sentiment analyser do the following:
1. Go to the URL : apps.twitter.com and login using your twitter account credentials.
2. Click on "Create New App". Fill in the necessary details to continue.
3. Once the app is created successfully, click on the Keys and Access Tokens tab.
4. Scroll down to find an option to create access token.
5. Once done. copy the consumer key, consumer secret, access token, access secret in the dataminer.py script.
